# 剪辑入门笔记

> 这是视频剪辑学习板块的示例。演示如何在网站里嵌入视频。

## 嵌入 B 站视频（推荐）

视频传到 B 站，网站只嵌入播放器，保持轻量：

<iframe src="//player.bilibili.com/player.html?bvid=BV1xx411c7mD"
        width="100%" height="500" allowfullscreen></iframe>

把上面的 `bvid` 换成你自己视频的 BV 号即可。

## 学习笔记

- 剪辑软件：剪映 / Premiere / DaVinci
- 基础流程：导入 → 剪切 → 调色 → 配乐 → 导出
